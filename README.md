# FreePigBro - 基于HanLP的聊天记录词频统计程序

---

这是一个非常简单的词频统计程序，用户需要从聊天记录导出工具[QQ Chat Exporter](https://github.com/shuakami/qq-chat-exporter)中导出群组聊天记录，并将文件放在`data`目录下，FreePigBro会自动地提取并进行分词，在控制台输出结果。

本程序来源于本人的一个小想法，花了差不多一天时间实现了一下，特别鸣谢以下项目：

[HanLP: Han Language Processing - 面向生产环境的多语种自然语言处理工具包](https://github.com/hankcs/HanLP)

[QQ Chat Exporter - 一款功能强大的 QQ 聊天记录导出工具](https://github.com/shuakami/qq-chat-exporter)

## 使用方式

1. 下载提供的压缩包并解压。

2. 将需要统计的聊天记录json文件放至`data`目录下。

3. 双击start.bat，根据引导获得数据。
